FCKEditor Plugins

PLUGIN KFM - Filemanager

If you want to be able to manage and edit different files (graphics and text) from FCKeditor, upload the plugin KFM to your XOOPS Cube Legacy installation: 

/common/fckeditor/editor/plugins/ "kfm"

Open and edit fckeditor configuration file, fckconfig.js
add the following code to the last line : 

FCKConfig.Plugins.Add('kfm');

Author website and documentation

http://kfm.verens.com/

PLUGIN GoogleMaps

If you want to be able to add a Google Map to a page that you are editing with FCKeditor upload the plugin Googlemaps to your XOOPS Cube Legacy installation:

/common/fckeditor/editor/plugins/ "googlemaps"

Edit fckeditor configuration file (fckconfig.js) and add the following code to the last line, separating available languages with comma 

FCKConfig.Plugins.Add( 'googlemaps', 'de,en,es,fr,it,jp') ;
